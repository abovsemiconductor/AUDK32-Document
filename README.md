# Documentation

Documentation package for the ABOV AUDK32 Software Development Kit (SDK).

This package contains user guides, peripheral descriptions, API references, release notes, and additional development resources for the AUDK32 platform.

## Contents

| File / Directory | Description |
|---|---|
| `Chipset_Description.md` | Overview and description of supported chipset features |
| `Doxygen/` | Generated API reference documentation |
| `Doxygen/AUDK32.chm` | Compiled HTML Help (CHM) format API documentation |
| `Doxygen/html/` | HTML format API documentation |
| `Release_Note.md` | SDK release history and change information |
| `STD10-HB_AUDK32 CLI Example_Eng.pdf` | CLI example user guide |
| `STD10-HB_AUDK32 Guide_Eng.pdf` | AUDK32 user guide |

## Documentation Overview

### User Guide

Main SDK usage guide:

```text
STD10-HB_AUDK32 Guide_Eng.pdf
```

Contents typically include:

- SDK introduction
- Development environment setup
- Project structure
- Build process
- Example usage
- Programming guide

Recommended as the starting point for new users.

### CLI Example Guide

Command-line example guide:

```text
STD10-HB_AUDK32 CLI Example_Eng.pdf
```

Includes:

- CLI command usage
- Example scenarios
- Reference application behavior

### API Reference

API documentation generated using Doxygen.

Available formats:

#### HTML

```text
Doxygen/html/index.html
```

Open `index.html` in a web browser.

#### CHM

```text
Doxygen/AUDK32.chm
```

Open using a CHM-compatible viewer.

Contents include:

- HAL APIs
- Peripheral APIs
- Function descriptions
- Structures
- Macros
- Enumerations

### Chipset Description

Chipset-specific information:

```text
Chipset_Description.md
```

Includes:

- Device overview
- Feature description
- Peripheral summary
- Device characteristics

### Release Notes

SDK version and update history:

```text
Release_Note.md
```

Includes:

- Added features
- Improvements
- Fixed issues
- Known limitations

## Recommended Reading Order

For first-time users:

1. `STD10-HB_AUDK32 Guide_Eng.pdf`
2. `Chipset_Description.md`
3. `Release_Note.md`
4. `Doxygen/html/index.html`

For application development:

1. SDK Guide
2. Example applications
3. API reference
